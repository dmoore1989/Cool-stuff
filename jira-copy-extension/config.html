Jira Copy Extension - Settings

How to get your Jira API Token:

  1. Go to Atlassian Account Settings
  2. Click "Create API token"
  3. Give it a label (e.g., "Jira Copy Extension")
  4. Copy the generated token and paste it below

Note: API tokens are stored in browser session storage and cleared when you close the browser.

Source Jira (Where to copy FROM)

The base URL of your source Jira instance
Your email address for the source Jira account
API token for authentication (stored in session storage)

Destination Jira (Where to copy TO)

The base URL of your destination Jira instance
The project key where tickets will be created (e.g., PROJ, ENG)
Your email address for the destination Jira account
API token for authentication (stored in session storage)
